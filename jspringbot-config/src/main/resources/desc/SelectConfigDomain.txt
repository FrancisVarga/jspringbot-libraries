Switch the currently selected domain configuration.

Example:

_classpath:domain1.properties_

| property1=domain1 property1 value

_classpath:domain2.properties_

| property1=domain2 property1 value

_usage_

| `Select Config Domain` | domain1                      |           |
| ${domain1Property1}=   | `Get Config String Property` | property1 |
| `Select Config Domain` | domain2                      |           |
| ${domain2Property1}=   | `Get Config String Property` | property1 |

=>

| ${domain1Property1} = "domain1 property1 value"
| ${domain2Property1} = "domain2 property1 value"
