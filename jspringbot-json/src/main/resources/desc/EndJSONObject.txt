Ends the last started JSON Object.

Example:

| `Start JSON Object`            |         |
| `Add JSON Object String Item`  | name    | config1
| `Add JSON Object Integer Item` | timeout | 1000
| `Add JSON Object Boolean Item` | enabled | true
| `End JSON Object`              |

{{Get Created JSON String}}

| {
|    "name": "config1"
|    "timeout": 1000
|    "enabled": true
| }
